MAMP mac集成环境套装,配置文件已经数据库储存位置。


经过多时的查找终于找到了`MAMP`在mac上面的储存位置：  
`/Library/Application Support/appsolute/MAMP PRO/`  
在Mac中进入该目录需要添加`\`转义空格  
`cd /Library/Application\ Support/appsolute/MAMP\ PRO/`

MAMP所用的配置文件、数据库文件都在该目录，包括phpMyAdmin
